## Project Description
This repository is created for Assignment #2 of SE4002.  
The purpose of this project is to demonstrate Agile workflow using Jira and GitHub integration.

The assignment covers:

- Jira Scrum Project Creation
- Epics and Backlog Management
- Sprint Planning
- GitHub Repository Setup
- Jira–GitHub Integration
- Branching using Jira Issue Keys
- Commits linked to Jira Issues
- Pull Requests and Sprint Tracking

## Tools Used
- Jira Software
- GitHub
- Git (basic version control)

## Workflow Summary
Each Jira issue has its own GitHub branch.  
Commits include Jira issue keys to enable automatic linking between Jira and GitHub.  
Pull requests are created and merged to track sprint progress.

Example:

Branch:
SPM-3-user-login

Commit:
SPM-3 Implement user login logic

## Objective
To understand how Agile planning in Jira connects with real development work in GitHub.
